# Quick Start
## Prerequisites
* make
    * macOS
      ```bash
      brew install make
      ```
    * Windows
      ```bash
      choco install make
      ```
    * Linux
        * Should be built into the system

##  Installation

1. Clone or download zip of git repo.
2. Open terminal, and ```cd``` to ```/src/cmake-build-release```
3. Run ```make```.
4. All done!

> If you want to use Qi globally, simply add it to your system **PATH**.

## Commands
- Open a REPL
```bash
./qi
```
- Run a file
```bash
./qi path_to_file.qi
```
